## ⚡ Quick Start (3 Steps)

### Step 1: Install Dependencies (2 minutes)
```bash
npm install
```

### Step 2: Create Environment File (1 minute)
```bash
# Windows
copy .env.example .env

# Mac/Linux  
cp .env.example .env
```

### Step 3: Run Development Server (30 seconds)
```bash
npm run dev
```

**Visit:** http://localhost:3000
